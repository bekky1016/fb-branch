## Fizzbuzz란?

1부터 16까지의 양의 정수에 대해그 임의의 정수 i가 만약
- 3의 배수라면 'fizz' 출력
- 5의 배수라면 'buzz' 출력
- 15의 배수라면 'fizzbuzz' 출력
- 나머지 모든 경우 i를 그대로 출력
